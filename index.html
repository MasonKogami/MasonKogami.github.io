Projects

Eventure - an Eventbrite inspired clone. Users can browse events, host events, and get tickets. Eventure is built with a React/Redux frontend, a Python/Flask backend, and supported by a PostgreSQL database. Future implementation includes: likes, Google Maps, AWS S3 images, and event categories.

Jello - a Trello inspired clone is a simple, easy-to-use collaboration tool that enables you to organize projects and everything related to them into boards. Jello is built with a React/Redux frontend, a Python/Flask backend, and supported by a PostgreSQL database.

SnackOverflow - a StackOverflow inspired clone with a theme of all things edible. Users can ask questions, post answers, and view their user profile page. SnackOverflow is built with a JavaScript/Pug frontend, aNodeJs/Express backend, and supported with a PostgreSQL database. Future implementations include: a light/dark mode toggle, and comment threads.
